Export Options

This document lists the parameters on the export window, and their descriptions.

Open Export Dir When Done: When checked, the output directory will be open upon export completion.

Export Textures: When checked, an attempt will be made to export all (readable) textures.

Generate Materials: When checked, the accompanying .MTL materials library will be generated for the OBJ file.

Material Instance Percautions: When checked, material instances will also contain their instance ID in the material name

Material Name From Texture Name: When checked, material names will be the name of the main texture, rather than the actual material name. If there is no main texture assigned then the original material name is used

Split Type: OBJ split type. Split types are: By Mesh, By Submesh, None

Selected Only: When checked ,only selected objects will be exported

Apply Scale: When checked, the scale of models will be taken into account on export

Apply Position: When checked, the position of models will be taken into account on export

Apply Rotation: When checked, the rotation of models will be taken into account on export
